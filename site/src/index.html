kopio.js

What does it do?

kopio.js is a light solution with no dependencies for doing copyable links/text for websites. It's powered by HTML5 data attributes + execCommand and createRange APIs.

Instantiate

The Kopio constructor has an optional string parameter that has to be a valid for use with the querySelector. The default selector is .kopio. All elements that match the query will have click event listeners added to them.


import Kopio from 'kopio.js';

new Kopio();
// or
new Kopio('.custom');

Events

The Kopio object emits only two events success and error. Both events pass the event trigger as a parameter for the callback.


const kopio = new Kopio();
kopio.on('success', (trigger) => {

});

kopio.on('error', (trigger) => {

});

Example

Simple example where the user can see when the copying is done.

COPY

The code used for the example:


const kopio = new Kopio();
kopio.on('success', (trigger) => {
  const el = document.createElement('div');
  const text = document.createTextNode('Copied');
  el.classList.add('notify');
  el.appendChild(text);
  trigger.appendChild(el);
  setTimeout(() => {
    el.remove();
  }, 500);
});
